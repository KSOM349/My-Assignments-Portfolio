Problem: Discount Calculation Error

Buggy Code:

def calculate_discount(price, discount_percent):
    discount = price * discount_percent
    final_price = price - discount
    return final_price

print(calculate_discount(100, 20))  # Expected: 80, Actual: -1900

Fixed Code:

def calculate_discount(price, discount_percent):
    discount = price * (discount_percent / 100)
    final_price = price - discount
    return final_price

print(calculate_discount(100, 20))  # Result: 80.0
